Social Media App

As well as the first one, this project was developed for a 2nd year subject named Human-Computer Interfaces.

The aim of this project was creating a simple application in order to allow the user to access to all their social media and other web pages from as single intutitive application. The user is allowed to add and remove applications as they wish.

This project was focused on learning basics of Application Developement. Furthermore we were asked to make it as intuitive as possible (but not elegant).

The application has been developed in Java using NetBeans to create and edit the interface.
